    Set round corners using a mask layer. This method of adding round corners are useful if want only few corners to be rounded (e.g.: TopLeft and TopRight corners).

    WARNING: Note that if the bounds of the view change, the mask layer must be recomputed. So its a good idea to call this method from your layoutSubviews() method.
